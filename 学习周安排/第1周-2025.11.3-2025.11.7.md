# 第1周学习安排（2025.11.3 — 2025.11.7）

说明：每日任务与验收严格对齐各专题的「按周迭代学习排期」第1周条目；产出沉淀在对应专题目录与当日复盘。

## 周一（Go：环境与骨架）
- 来源：`go/按周迭代学习排期.md` 第1周
- 目标：安装 Go，初始化 `go-expert` 仓与工程骨架。
- 实战任务：
  - `go mod init`，搭建目录：`cmd/api`、`cmd/worker`、`cmd/cli`、`internal/{handler,service,repo,domain}`、`pkg/{collections,workerpool}`、`scripts/`。
  - 编写基础 main：API 启动与健康探针；Worker 池示例（可取消）。
  - PowerShell 构建/测试脚本：`scripts/build.ps1`、`scripts/test.ps1`。
- 验收标准：`go build`、`go test` 通过；`-race` 无数据竞争；脚本可一键构建与运行。
- 产出物：仓库骨架、README、脚本；当日复盘 `docs/day-2025-11-03.md`。

## 周二（MySQL：安装与基础 SQL）
- 来源：`mysql/按周迭代学习排期.md` 第1周
- 目标：安装 MySQL 8.0，完成基础表与数据准备。
- 实战任务：
  - 启动 MySQL（本机或 Docker），准备连接脚本。
  - 创建 `orders`、`order_items`、`inventory`（主键/唯一/外键/审计字段/生成列）。
  - 导入 100 万行测试数据（PowerShell 或 Python 生成器）。
  - 完成基础 CRUD 与典型查询（覆盖索引可用）。
- 验收标准：数据导入稳健；基础查询正确；初步 `EXPLAIN` 合理。
- 产出物：迁移 SQL、数据生成脚本、连接配置；当日复盘 `docs/day-2025-11-04.md`。

## 周三（Redis：安装与基础操作）
- 来源：`redise/按周迭代学习排期.md` 第1周
- 目标：安装 Redis 6/7，完成核心数据结构与基准测试。
- 实战任务：
  - 本地/容器启动 Redis；编写连接与基础操作脚本（`SET/GET/HMSET/LPUSH/SADD/ZADD`）。
  - 设计键空间命名规范（前缀/层级/TTL）。
  - 运行 `redis-benchmark`，生成初步报告（QPS、P50/P99）。
- 验收标准：脚本可复用；基准报告生成；键空间规范文档齐备。
- 产出物：命令脚本、键空间设计说明、基准报告；当日复盘 `docs/day-2025-11-05.md`。

## 周四（消息队列：环境搭建与最小可运行）
- 来源：`消息队列/按周迭代学习排期.md` 第1周
- 目标：本地启动 Kafka 与 RabbitMQ，完成生产/消费 Hello World。
- 实战任务：
  - 启动 Kafka 与 RabbitMQ（本机或 Docker）。
  - 创建 `topic: orders_v1` 与 `queue: orders_q`。
  - 编写 Java Producer/Consumer（可用 Spring Boot），发送/消费 JSON 消息。
  - 发送 1 万条消息，观测基础指标（吞吐/延迟）。
- 验收标准：消息无丢失；基础指标可见（吞吐/延迟）。
- 产出物：`spring-boot-mq-demo` 初始仓、启动脚本、指标截图/报告；当日复盘 `docs/day-2025-11-06.md`。

## 周五（机器学习：环境与 EDA）
- 来源：`机器学习/机器学习.md`（第1周建议）
- 目标：准备 Python/Notebook 环境，完成数据加载与探索性分析。
- 实战任务：
  - 安装 Python/Conda，配置 Jupyter；安装 NumPy/Pandas/Matplotlib/Seaborn。
  - 选择公开数据集（Titanic/房价等），完成数据加载/清洗/EDA 与可视化。
  - 输出初步建模方向与下一步特征工程计划。
- 验收标准：完成一个 Notebook（含清洗、统计与图表）；结论明确。
- 产出物：`notebooks/EDA-2025-11-07.ipynb`、可视化截图；当日复盘 `docs/day-2025-11-07.md`。
